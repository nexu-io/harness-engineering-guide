# 商业平台

商业化的 Harness 和托管 Agent 平台。

| 平台 | 类型 | 模型支持 | 核心特色 |
|------|------|---------|---------|
| [Claude Code](https://docs.anthropic.com/en/docs/claude-code) | 一体化 Harness | Claude | 512K 行 Harness，深度 IDE 集成 |
| [Codex](https://openai.com/codex) | 云端 HaaS | GPT | 云端沙箱，并行任务 |
| [Cursor](https://cursor.sh) | IDE 原生 Harness | 多模型 | AI-first IDE，Tab 补全 |
| [Windsurf](https://windsurf.ai) | IDE 智能体 | 多模型 | 编辑器内 Flow 智能体 |
| [Devin](https://devin.ai) | 云端智能体 | 专有模型 | 自主软件工程师 |
| [CREAO](https://creao.ai) | 智能体平台 | 多模型 | 企业级 AI 超级智能体 |

---
