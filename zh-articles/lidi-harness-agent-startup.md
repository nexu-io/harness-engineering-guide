# 李笛带队 Harness 智能体公司获李开复陆奇重仓，4个月完成2轮融资

> **Original**: [量子位](https://www.qbitai.com/2026/04/401065.html) · 西风 · 2026-04-14
> **Category**: Industry / Funding

---

## Overview

据量子位报道，一家由李笛带队的 Harness 智能体创业公司在 4 个月内完成了两轮融资，储备了 3-5 年的资金弹药。值得关注的是，李开复和陆奇同时重仓了这家公司，两位 AI 领域最具影响力的华人投资人 / 企业家罕见地押注同一标的。

## Why This Matters for Harness Engineering

这笔融资的信号意义远大于金额本身：

**1. Harness 已从技术概念变成投资赛道**

"Harness Engineering" 这个术语在 2026 年初由 OpenAI 的 Codex 团队正式提出，仅 2-3 个月就从技术博客概念演变为有大额资本下注的创业赛道。这是 AI Agent 基础设施领域最快的从概念到融资的周期之一。

**2. 顶级投资人的共识**

李开复（创新工场 / 零一万物）和陆奇（奇绩创坛）分别代表了 AI 应用层和基础设施层的两种投资视角。两人同时看好同一家 Harness 公司，说明市场对 "Agent Runtime" 这个基础设施层已经形成了强共识。

**3. 3-5 年粮草 = 长期基础设施赌注**

不是 12 个月的快钱，而是 3-5 年的储备。这意味着投资人和团队都认为 Harness 不是一个风口项目，而是需要长期深耕的基础设施层。

## The Landscape Context

这家公司的出现进一步验证了 Harness Engineering 生态正在快速分化：

| 层级 | 代表项目 | 定位 |
|------|---------|------|
| **开源框架** | OpenClaw, Cline, Aider, OpenCode | 社区驱动的 Harness Runtime |
| **商业平台** | Devin, Cursor, Windsurf | 面向开发者的商业 Agent IDE |
| **基础设施** | 李笛团队（本文）, CREAO AI | 面向企业的 Agent 基础设施 |
| **生态标准** | Harness Engineering Guide | 开源知识体系 |

李笛此前曾领导微软小冰（后独立为小冰公司），在对话式 AI、多模态交互领域有深厚积累。从对话 AI 转向 Agent Harness 基础设施，是一个自然的技术演进路径。

## Key Takeaway

> Harness Engineering 不再只是一种工程实践——它正在成为一个有独立融资逻辑的创业赛道。当李开复和陆奇同时下注时，通常意味着这个方向已经从 "early signal" 变成了 "inevitable trend"。
