# 基础论文

塑造 Harness Engineering 和 AI Agent 运行时领域的关键论文。

## 智能体架构

- **ReAct: 在语言模型中协同推理与行动** (2022)
  Yao 等 | [论文](https://arxiv.org/abs/2210.03629)
  推理+行动循环，现代 Agent Harness 的核心基础。

- **Toolformer: 语言模型可以自学使用工具** (2023)
  Schick 等 | [论文](https://arxiv.org/abs/2302.04761)
  Agent Harness 中工具调用的理论基础。

- **Generative Agents: 人类行为的交互式拟像** (2023)
  Park 等 | [论文](https://arxiv.org/abs/2304.03442)
  持久化智能体身份的记忆架构。

- **新兴 AI Agent 架构全景：推理、规划与工具调用** (2024)
  Masterman 等 | [论文](https://arxiv.org/abs/2404.11584)
  智能体架构模式的综合综述。

## 记忆 & 上下文

- **MemGPT: 迈向 LLM 即操作系统** (2023)
  Packer 等 | [论文](https://arxiv.org/abs/2310.08560)
  受 OS 内存层级启发的虚拟上下文管理。

- **Reflexion: 基于语言反馈的强化学习智能体** (2023)
  Shinn 等 | [论文](https://arxiv.org/abs/2303.11366)
  自我反思作为记忆和学习机制。

## 多智能体系统

- **AutoGen: 通过多智能体对话实现下一代 LLM 应用** (2023)
  Wu 等 | [论文](https://arxiv.org/abs/2308.08155)
  多智能体对话框架的设计模式。

- **AgentVerse: 促进多智能体协作** (2023)
  Chen 等 | [论文](https://arxiv.org/abs/2308.10848)
  多智能体系统的协调模式。

## Harness Engineering（新兴方向）

- **Harrison Chase: "你的 Harness，你的记忆"** (2026)
  [博客](https://blog.langchain.dev/) | [推文](https://x.com/hwchase17)
  将 "Harness Engineering" 推入主流话语的文章。

- **steipete: OpenClaw 严格模式与 Harness 可插拔性** (2026)
  [推文](https://x.com/steipete)
  运行时级约束 vs 提示词级约束。
