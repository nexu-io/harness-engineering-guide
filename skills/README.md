# Skills

Reusable skills for Harness Engineering workflows. Each skill is a self-contained package with a `SKILL.md` and optional scripts.

## Available Skills

| Skill | Description |
|-------|-------------|
| [abuse-hunter](./abuse-hunter/) | SaaS 批量注册盗刷排查 — 6 步排查流程 + 12 分制评分 + SQL 模板 + Python 离线分析 |

## How to Use

1. Copy the skill folder into your agent workspace `skills/` directory
2. The agent will automatically detect and use the skill when relevant tasks arise
3. Or reference the `SKILL.md` directly for manual analysis workflows
